# Theme Samples

Short source files for checking how the active color theme renders common
syntax groups across different languages.

Open the files in your editor and compare:

- keywords
- types
- functions
- strings
- numbers
- comments
- shell directives like `export`

Suggested files:

- `bash.sh`
- `fish.fish`
- `javascript.js`
- `lua.lua`
- `python.py`
- `rust.rs`
- `toml.toml`
- `yaml.yaml`
