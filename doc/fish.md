# Fish Shell

## Keybindings

### History Navigation

- `Ctrl+r` - Search command history
- `Ctrl+p` - Move to the previous command in history
- `Ctrl+n` - Move to the next command in history

### Cursor Movement

- `Ctrl+a` - Move cursor to beginning of line
- `Ctrl+e` - Move cursor to end of line
- `Ctrl+f` - Move cursor forward one character
- `Ctrl+b` - Move cursor backward one character

### Editing

- `Ctrl+k` - Delete text from cursor to end of line
- `Ctrl+u` - Delete text before cursor
- `Ctrl+w` - Delete word before cursor
- `Ctrl+d` - Delete character under cursor
- `Ctrl+h` - Delete character before cursor
- `Ctrl+z` - Undo last change
- `Ctrl+y` - Paste last deleted text
- `Ctrl+t` - Swap last two characters before cursor

### Screen & Process Control

- `Ctrl+l` - Clear screen
- `Ctrl+z` - Suspend current process (send to background)
- `Ctrl+c` - Kill current running process

### Job Control Commands

- `fg` - Bring most recently suspended process to foreground
- `bg` - Resume suspended process in background

### External Editor

- `Alt+e` - Edit current command line in external editor
- `Alt+v` - Edit current command line in external editor
