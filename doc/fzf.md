# FZF (Fuzzy Finder)

## fzf.fish Plugin Keybindings

### Fuzzy Search

- `Ctrl+Alt+f` - Search file in current directory
- `Ctrl+Alt+l` - Search commit in git log
- `Ctrl+Alt+s` - Search file in git status
- `Ctrl+Alt+p` - Search process
- `Ctrl+r` - Search command history
- `Ctrl+v` - Search environment variable

### Navigation in FZF

- `Ctrl+p` - Move to previous entry
- `Ctrl+n` - Move to next entry
- `Tab` - Select multiple entries
- `Enter` - Confirm selection
- `Esc` - Cancel/exit
