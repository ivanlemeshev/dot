# Setup Ubuntu

## Run setup script

```bash
./setup-ubuntu.sh
```

## Run tmux and install plugins

```bash
tmux
```

Press keyboard shortcut:

```text
Ctrl+Space I
```

## Run neovim and install plugins

```bash
vi
```

Wait for plugins to install.

Enable GitHub Copilot:

```text
:Copilot auth
```
