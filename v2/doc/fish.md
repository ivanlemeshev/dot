# Fish Shell

## Basic Shortcuts

- `Ctrl+p` - Previous command in history
- `Ctrl+n` - Next command in history
- `Ctrl+a` - Move cursor to beginning of line
- `Ctrl+e` - Move cursor to end of line
- `Ctrl+f` - Move cursor forward one character
- `Ctrl+b` - Move cursor backward one character
- `Ctrl+k` - Delete text from cursor to end of line
- `Ctrl+u` - Delete text before cursor
- `Ctrl+w` - Delete word before cursor
- `Ctrl+d` - Delete character under cursor
- `Ctrl+l` - Clear screen
- `Ctrl+c` - Cancel current command
- `Ctrl+z` - Suspend current process

## Editing

- `Alt+e` - Edit current command in external editor
- `Alt+v` - Edit current command in external editor

## Process Control

- `Ctrl+z` - Suspend process and send to background
- `fg` - Bring suspended process to foreground
- `bg` - Resume suspended process in background
